# Assets

Esta carpeta contiene los íconos e imágenes del splash de la app.

Reemplaza estos placeholders por los assets reales de tu TEG:

- `icon.png` — 1024×1024, fondo opaco. Es el ícono que ve el usuario en el home.
- `adaptive-icon.png` — 1024×1024, foreground PNG con transparencia (Android).
- `splash.png` — 1284×2778 recomendado, se muestra mientras carga la app.

Expo genera automáticamente las variantes para cada densidad de pantalla a
partir de estos archivos cuando corres `expo prebuild`.
